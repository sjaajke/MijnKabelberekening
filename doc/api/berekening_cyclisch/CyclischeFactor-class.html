CyclischeFactor

CyclischeFactor class

Cyclische reductiefactor M per NEN IEC 60583-1:2002 / NEN IEC 60287-2-1:2023. Alleen van toepassing bij kabels in de grond (leggingswijze D1 of D2).

De factor M ≥ 1 geeft aan hoeveel meer stroom een kabel kan voeren als de belasting niet continu maximaal is.
